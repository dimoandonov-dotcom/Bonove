Обработка на касови бонове

Бонове до лимит

Качи PDF или снимки с бонове. Приложението разчита крайната сума на всеки бон, събира ги по ред и намира до кой бон натрупаната сума достига лимита. Сборовете се смятат тук, точно.

API ключ

Вземи ключ от console.anthropic.com → API Keys. Зареди кредит от Billing и задай месечен лимит на харчене за сигурност.
